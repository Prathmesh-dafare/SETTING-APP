# Exam Seating Management System

A full-stack web application for managing exam seating arrangements efficiently.

## 🚀 Features

- Admin Login Authentication
- Student Management
- Room Management
- Exam Scheduling
- Automatic Seating Arrangement Generation
- PDF Export for Seating Plans
- Dashboard Analytics
- Cloud Deployment

## 🛠 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python Flask

### Database
- MySQL (Railway)

### Deployment
- Render
- Railway
